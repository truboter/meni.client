# GitHub Copilot Preferences

## Правила работы с проектом

### Управление терминалами

- ✅ Закрывать терминалы после получения результата
- ✅ Использовать один терминал для последовательных команд
- ❌ Не множить терминалы без необходимости

### Перезапуск сервера

- ✅ После завершения внесения изменений в код **обязательно перезапустить** dev сервер
- Команды для перезапуска:
  ```bash
  pkill -f "vite.*7003"
  npm run dev:7003
  ```
  Или через VS Code задачу: "Start Frontend (Port 7003)"

### Стиль работы

- Минимизировать количество открытых терминалов
- Проверять ошибки после изменений
- Подтверждать успешность перезапуска сервера
